### Friend functions
  1. there us a feature in C++ called friend functions that allows a function to access the private and protected members of a class.
  2. Declared within the class but defined outside the class.
  3. Ususally, it has objects as arguments.
  4. Friend function is not a part of a class, it can be declared under private also.
  5. Although it is not a part of the class it has to be defined inside the class.
  6. If N classes share a friend function then it has to be declared in all N classes.
