### Static data member
  1. If we do not intialize a static variable then its value is 0.
  2. Memory of static data member is same for all objects.
  3. For normal data members memory is allocated differently for every variable.
  4. Static data members have to be intialised outside the class.
  5. **we cant Intialize a static data member inside a class in cpp.**

### Static member function
  1. ***Static member fucntions can access only static data members of the class.***
